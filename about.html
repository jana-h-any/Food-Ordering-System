About the Project

BTWEETY is a modern Food Ordering System designed to simplify the way users order their favorite meals online. The system allows users to browse restaurants, view menus, select meals, and place orders seamlessly. It aims to provide a user-friendly and fast experience while connecting customers with local restaurants.

Technologies Used

  • HTML, CSS, JavaScript for the web interface
  • Responsive design for different devices
  • Backend: Node.js / PHP / any backend if used
